# Redrob AI Candidate Ranking

## Overview
This project ranks candidates for the Redrob Senior AI Engineer position.

## Features
- Reads candidate profiles
- Scores candidates based on:
  - AI skills
  - Experience
  - GitHub activity
  - Recruiter response rate
  - Open-to-work status
- Produces Top100Candidates.xlsx

## Technologies
- Python
- Pandas
- OpenPyXL
